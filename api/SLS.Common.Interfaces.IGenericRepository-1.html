UpdateIgnoreNP(TEntity)

Updates the specified entity in the database while ignoring navigation properties. This method attaches the entity to the context if it is not already tracked, marks it as modified, and ensures that navigation properties are not marked as modified to avoid relationship changes. If the entity is already tracked, scalar properties are updated using SetValues, and navigation properties are handled to maintain consistency.
