Interface IProcessStatusService

Namespace
SLS.Common.Interfaces
Assembly
SLS.Common.Interfaces.dll

Interface zum Statuswesen. Implementation heißt: 'ProcessStatusService' und befindet sich in BusinessClasses. Wird für das Setzen von Status aus den FrameWork verwendet. Z.B.: Es können Status konfiguriert werden, die gesetzt werden wenn Berichte aus dem 'SLSReportManager' gedruckt/exportiert/archivert oder per E-Mail versendet werden.
